1.ทำการโหลดโปรแกรม Xampp สามารถโหลดได้ตามลิ้งก์ https://www.apachefriends.org/index.html
2.เมื่อโหลดแลพทำการติดตั้งเสร็จสิ้น ให้ทำการโหลดไฟล์ Project.php และ word.txt นำไปติดติ้งไว้ที่ C:\xampp\htdocs เพื่อทำการ Run ไฟล์
3.เปิดโปรแกรม Xampp ทำการกดปุ่ม Start ในส่วนของ Apache 
4.หลังจากนั้นกดปุ่ม Admin
5.เมื่อทำการกดปุ่ม Admin จะทำการเปิดหน้าต่าง Browser เพื่อ Run Project.php และจะแสดงผลใน Folder ที่ทำการติดตั้งไฟล์
